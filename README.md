# Microprocessor & Microcontroller Lab Works

This repository contains academic lab programs related to Microprocessor (8086 Assembly) and Microcontroller (Arduino-based systems).



##  Repository Structure

###  Microprocessor
Contains 8086 Assembly language programs including:
- Even/Odd number checking
- Addition and subtraction
- Largest number finding
- Basic input/output operations
- Loop-based arithmetic operations

###  Microcontroller
Contains Arduino-based embedded system programs including:
- LED blinking and digital I/O control
- Servo motor control (360°)
- LCD display with keypad password system
- Ultrasonic sensor distance measurement
- Serial monitor based input/output tasks



##  Tools Used

### Microprocessor
- 8086 Emulator / MASM / TASM
- Assembly Language (x86)

### Microcontroller
- Arduino UNO (simulated / conceptual)
- Arduino IDE
- C/C++ (Embedded Programming)



##  Hardware Note

Some microcontroller experiments were performed using simulation or previously completed lab setups. Physical hardware (Arduino board, sensors, display modules) may not be available for all experiments.


##  Objectives

- Understand low-level assembly programming (Microprocessor)
- Understand embedded system design (Microcontroller)
- Implement arithmetic, logic, and control-based programs
- Learn sensor integration and real-time system concepts


##  Learning Outcomes

- 8086 Assembly programming basics
- Stack, loop, and conditional logic in assembly
- Arduino digital and analog control
- Sensor and actuator interfacing
- Embedded system problem solving
