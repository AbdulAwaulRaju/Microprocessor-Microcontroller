# Microprocessor & Assembly Language (8086)

This repository contains Microprocessor (8086) assembly language lab works and assignments.  
All programs are written in MASM/TASM style assembly language.

---

##  Contents

### Assignment
- Even/Odd number counter (10 inputs)
- Largest and Second Largest number finder with sum

---

##  Topics Covered

- Input/Output operations
- Looping in Assembly
- Conditional branching (if-else logic)
- Arithmetic operations
- Number comparison logic

##  Tools Used
- 8086 Microprocessor Architecture

##  Purpose

This repository is created for academic lab submission of Microprocessor and Assembly Language course.
