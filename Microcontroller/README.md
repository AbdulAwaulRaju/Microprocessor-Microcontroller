# Microcontroller & Embedded Systems Lab

This repository contains Arduino-based microcontroller lab works completed as part of academic coursework.

##  Contents

The repository includes the following experiments:

- LED Blinking and Basic I/O operations
- Even/Odd number checking using Arduino logic
- Basic arithmetic operations using Serial Monitor
- Servo Motor (360° continuous rotation control)
- LCD Display with Keypad-based Password System
- Ultrasonic Sensor based Distance Measurement


## Tools & Platform

- Arduino UNO (simulated / theoretical implementation)
- Arduino IDE
- C/C++ based embedded programming
- Serial Monitor for input/output simulation


##  Hardware Note

Due to unavailability of physical components (such as Arduino board, servo motor, LCD display, keypad, ultrasonic sensor, etc.), all experiments have been implemented using code-based logic and simulation approach.


##  Practical Implementation

No physical images are included as the experiments were performed in a simulated or previously completed lab environment.


##  Learning Outcomes

- Understanding of microcontroller programming concepts
- Working with digital I/O pins
- Sensor integration logic
- Motor control using Arduino
- Display and input handling systems
